Tagu sarasas

  • p - paragraph - paragrafas skirtas skirtas rasyti pastraipai
  • h1-h6 - Heading1-Heading6 - antrastes skirtos rasyti pavadinimams ir tarpusavyje skiriasi teksto dydziu
  • img - Image - Paveiksliukas. Naudojamas kartu su alt ir src atributais. Kiti pagalbiniai
  • a - Hypertext Anchor (link) - Nuoroda
  • hr - Thematic Break (horizontal line) - tematinis/horizontalus luzis, kuris sukuria eilutes luzi su linija
  • br - Line Break - Eilutes luzis
  • ul - Unordeder List - Nerikiuotas sarasas
  • ol - Ordered List - Rikiuotas sarasas
  • li - List Item - Saraso elementas (skirtas rikuotam ar nerikiuotam sarasui)
  • meta - skirtas nurodyti psl meta duomenis
  • title - Skirtas nurodyti puslapio tab'o pavadinima

Atributu sarasas

  • lang - language - Nurodo kokia kalba vyrauja tame tag'e
  • charset - Character Set - Simbolliu rinkinys, kuris nurodo kokia simboliu biblioteka naudoti
  • src - Source - Saltinis, nurodantis faila (kelias iki failo)
  • alt - Alternative Text - Alternatyvus tekstas paveikslio neuzkrovimo atveju
  • Width - Nurodo elemento ploti
  • Height - Nurodo elemento auksti
  • href - Hyper link reference - nuoroda i puslapi arba faila
  • Target - Nurodo kur atidaryti nuoroda (kokiame puslapyje)

Klaviaturos trumpiniu sarasas

  • CTRL S - issaugoja faila
  • CTRL C - kopijuoja pazymeta info
  • CTRL X - iskerpa pazymeta info
  • CTRL V - iklijuoja kopijuota/iskirpta info
  • CTRL Z - atsaukia paskutini veiksma
  • CTRL F - atidaro paieskos sistema
  • TAB - Tabliuoja pazymeta koda i desine
  • SHIFT TAB - Tabliuoja pazymeta koda i kaire
  • CTRL A - Pazymi viska
  • CTRL / - Komentuoja pazymetas eilutes
  • CTRL Y - Atsaukia CTRL Z veiksma/veiksmas i prieki
  • ALT Z - Word Wrap
  • CTRL R - Duoda pasirinkti ka atidaryti is failu aplankalu istorijos
  • ALT + skaiciuKombinacija - Paraso atitinkama kombinacijai priskirta simboli
  • CTRL SHIFT L - Pazymi visus vienodus zodzius/sakinius, taip pat galima juos keisti
  • CTRL SHIFT K - Istrina pazymeta/as eilutes
  • CTRL SHIFT i/j arba f12 - Atidaro Developer Tools (narsykleje)
  • CTRL D - Pazymi viena toki pati zodi/ius einanti toliau kode
  • END - Nukels kursoriu i eilutes gala
  • HM (home) - Nukels kursoriu i eilutes pradzia
  • SHIT ALT arrowUP/DOWN - Dublikuoja pazymeta/as eilute/es aukstyn ar zemyn
  • CTRL ENTER - Nauja eilutė
  • Windows Tab - sukuria naują destopo langą
  • CTRL ArrowLeft/ArrowDown - Šokinėja per žodžius
  • SHIFT Pažymi ir vaikšto per tekstą
  • CTRL T - Atidaro naują tabą naršyklėje
  • CTRL SHIFT T - Atidaro paskutinį uždarytą langą
