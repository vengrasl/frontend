Responsive Design

Responsive Design gali reaguoti į vartotojo įrenginį ir/arba jo plotį.

Esant skirtingiems ekranų dydžiams - galima pritaikyti skirtingus stilius naudojant @media query.

Breakpoint devices

  1. portrait phone
  2. table / landscape phone
  3. laptop
  4. desktop

Breakpoint width (px)

  1. <500px
  2. <678px
  3. <992px
  4. <1200px
  5. <1400px
  6. >1400px

Mobile First

  • Iš mažesnio ekrano ekrano dydžio yra lengvesnis perėjimas į didesnį (lengviau pridėti naują turinį nei atimti seną).
  • Koncentracija į turinį (dėl mažo vietos kiekio)
  • Didesnis kiekis vartotojų naudoja mob. aplikacijas (orientacija į didesnį vartotojų kiekį)
  • Google paieškos algoritmas geriau įvertina Mobile first fizainu kurtas svetaines

Naudojant Mobile First dizaino principą, pradedame nuo mažiausio ekrano ir judame link didesnių.
tam naudojame @media (min-width:_____px).
Naudojant @media (min-width: ____px) svarbu judėti nuo mažesnių dydžių į didesnius.
