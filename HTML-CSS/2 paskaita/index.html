Stilizavimo tagai (linijiniai)

  • b - bold - Pastorina tekstą
  • strong - Pastorinatekstą (taip pat atlieka semantinę prasmę)
  • i - Italic - Suteikia tekstui pasvyrimo stilių
  • em - emphasize - Suteikia tekstui pasvyrimo stilių (taip pat atlieka semantinę prasmę)
  • big - Padidina tekstą (pasenęs naudojimas)
  • small - Pamažina tekstą
  • sup - superscript - Pakelia tekstą į viršų ir jį sumažina
  • sub - subscript - Nuleidžia tekstą į apačią ir jį sumažina
  • span - Išskiria kažkokį elementą ar jo dalį

Lentelės

  • table - Lentelė
  • thead - Table Head - Lentelės viršus / pirmoji eilutė
  • tbody - Table body - Lentelės kūnas / duomenys (
  • tfoot - Table Foot - Lentelės apačia / paskutinė eilutė
  • tr - Table Row - Lentelės eilutės
  • th - Table Heading - Lentelės antraštė (stulpelio/eilutės pavadinimas
  • td - Table Data - Lentelės domuo
  • caption - lentelės pavadinimas
  • colgroup - Column Group - Stulpelių grupės
  • col - column - Stulpelis

Formos

Tag'ai

  • form - Forma
  • input - Įvestis
  • select - Pasirinkimas iš variantų
  • option - Pasirinkimo variantai
  • texterea - Didelio teksto kiekio įvestis
  • label - Etiketė prie įviesties/pasirinkimo
  • optgroup - Pasirinkimo variantų grupė
  • button - Mygtukas
  • fieldset - Formos laukas
  • legend - Formos lauko pavadinimas

Atributai

  • type - Input tag'o tipas
  • for - Nurodo kuriam įvesties/parinkties laukui yra priskirta etiketė (label)
  • id - Unikalus pavadinimas, į kurį nurodo for atributas
  • name - Įvesties lauko vardas, kuris bus perduotas back'ui, kartu su tos įvesties reikšme
  • cols - Stulpelių kiekis texterea elemente
  • rows - Eilučių kiekis texterea elemente
  • placeholder - Tekstas nusakantis ką rašyti įvesties lauke, kuris pradingsta pradėjus rašyti įvesties lauke
  • value - įvesties reikšmė, kuri perduoda back'ui
  • label - Priskiria colgroup'ui pavadinimą
  • selected - Numatytasis select tag'o viduje esantis pasirinkimo variantas
  • minlenght - Trumpiausia galima įvestis
  • maxlenght - Ilgiausia galima įvestis
  • min - Mažiausia leistina įvestis
  • max - Didžiausia leistina įvestis
  • step - Žingsnis, kuriuo didėja/mažėja skaičius
  • pattert - Reikalauja, kad įvesties laukelyje būtų parašytas nurodytas formatas
  • checked - Numatytasis radio arba checkbox pasirinkimas
