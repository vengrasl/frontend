Inline - linijinis

Linijinis stilius yra, kai elementui, kurį stilizuojame suteikiame stye atributą ir jo viduje aprašome stilius. Neturi perpaunaudijimo galimybės ir yra skirtas tik testavimui

Internal - vidinis

Vidinis stilius yra, kai HTML failo viduje, head tag'o dalyje pasirašoma style tag'ą ir tame tag'e rašome css kodą. Veikia visur, bet realiai naudojimai tik head' dalyje. Turi šiek tiek perpaunaudojimo galimybių (nes turi selektorius) ir dažniausiai naudijimas testavimui.

External - Išorinis

Išorinis stilius yra rašomas atskirame CSS faile ir yra prijungiamas pirie HTML failo naudojant link tag'ą. Turi visas reikalingas/naudojamas, yra plačiausiai naudojamas ir projekto variante, turi likti tik šis stiliaus rašymo būdas.

Basic Selectors

Element / Tag

Selektinami visi taga'ai, pagal nurodytą elementą.
Pvz.: h2 -> parinks visus h2 elementus esančius HTML'e.

Class

Klasė sukuriama kokiu tik norima logišku ir trumpu pavadinimu. Css'e Tam, kad nurodytume, jog kuriame klasę,sugalvoto pavadinimo pradžioje parašome tašką. HTML'e norint priskirti klasę elementui, tam elementui suteikiame atributą "class" ir jam priskiriame sugalvotą pavadinimą be taško.
pvz.:CSS'e .klasesVardas, o HTML'e < p class="klasesVardas"> Klasės turi daug su daug santykį. Vienas elementas gali turėti daugelį klasių ir viena klasė gali būti priskirta daugeliui elementų.

ID

ID sukuriame kokiu tik norime logišku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jog kuriame ID - sugalvoto pavadinimo pradžioje parašome groteles. HTML'e norint priskirti ID elementui, tam elementui suteikiame atributą "id" ir jam priskiriame sugalvotą ID pavadinimą be grotelių.
Pvz.: CSS'e: #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi santykį vienas su vienu. Vienas elementas gali turėti tik vieną ID ir vienas ID turėtų būti priskirtas tik vienam elementui.

colours

Names

Spalvos gali turėti vardus. Vardinių spalvų yra 140. Norint priskirti elementui spalvą, priskiri tos spalvos vardą.
Pvz.:red; blue; gree; lightsalmon; ...

RGB

RGB - Red Green Blue. Kiekvienos iš šių spalvų kiekio galima idėti nuo 0 iki 255 ir taip gauti bet kurią spalvą. Pvz.: rgb(0, 0, 0) - juoda; rgb(252, 0, 0) - raudona;...
RGBA - Red Green Blue Alpha. Alpha nurodo spalvos permatomumą ir jis gali būti nuo 0 iki 1 tūkstantąja paklaida. Pvz.: rgba(0, 255, 0, 0) - nematoma (žalia); rgba(0, 255, 0, 1) - pilnai matoma (žalia); ...

Hex

Hex - hexadecimal - šešioliktainis. Kiekvienos iš spalvos (RGB) galima nurodyti nuo 00 iki ff ir taip gauti bet kurią spalvą.
Skaičiavimas šešioliktainėje sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0c 0d 0e 0f 10 11 12 13...... 19 1a 1b ..... 1f 20 21 22 .... 9f a0 a1 a2 .... a9 aa ab ac ad ae af b0 ...... ff
Pvz.: #000000 - juoda; #ff0000 - raudona; ...
Trumpiniai: #000 - juoda; #f00 - raudona; ...
Spalvos matomumas su Hex: gale prisideda dar du šešioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas
Pvz.: #00ff0000 - nematoma (žalia); #00ff00ff - pilnai matoma (žalia)
Trumpiniai: #0f00 - nematoma (žalia); #0f0f - pilnai matoma (žalia); ...
