Vaikinis / Tėvinis elementas

Tėvinis elementas gali turėti daugelį vaikinių elementų.
Vaikinis elementas gali turėti tik vieną tėvinį elementą.

Paveldėjimas

Paveldėti stilius gali tik vaikinis elementas iš tėvinio elemento.
Paveldimi stiliai yra tie, kurie keičia elemento išvaizdą, ne būseną. Pvz.: color, font-size, font-family... Nepaveldimi stiliai yra tie, kurie keičia elemento būseną. Pvz.: width, height, display, border, position...
Net paveldėti stiliai, gali nebūti atvaizduoti dėl stilių prioritetų.

Numatytieji stiliai

Numatytieji stiliai yra tag'ams auto suteikti stiliai. Tarp skirtingų naršyklių, Numatytieji stiliai gali minimalistiškai keistis ir šitaip sugadinti svetainės vaizdą skirtingų naršyklių vartotojams
Tam, kad nebūtų problemų kuriant dizainą skirtingoms naršyklėms, tyra naudojami Reset arba Normalize CSS'ai.
Reset - resetina visus stilius (ištrina stilius)
Normalize - suvienodina visus stilius tarp naršyklių ir panaikina dalį kenkėjiškų stilių

CSS Prioritetai

  1. Inline Style - 1000
  2. ID - 100
  3. Class - 10
  4. Element - 1
  5. CSS points
  6. Css Order (in css code)
  7. Inheritence

Esant 2 selektoriams, kurie nusako to paties elemento vienodus stilius, yra atsižvelgiama į tų selektorių prioritetus. Aukščiau esantys prioritetai, turi pirmenybę.
