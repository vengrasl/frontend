Pseudo selektoriai

  • *
  • element
  • .class
  • #ID
  • selector1, selector2
  • selector1 selector2
  • selector1 > selecor2
  • element.class#id - elementas turintis nurodytą klasę arba klases ir nurodytą id
  • selector1 + selector2 - iš karto už selector1 einantis elementas (tame pačiame lygyje)
  • selector1 ~ selector2 - už selecotr1 visi einantys selector2 (tame pačiame lygyje)
  • [atribute] - atributas
  • [atributas=value] - atributas turintis nurodytą reikšmę
  • [atributas~=value] - atributas, kurio reikšmėje yra nurodytas žodis
  • [atributas*=value] - atributas, kurio reikšmėje yra nurodyta žodio dalis
  • [atributas^=value]
    • - atributas, kurio reikšmė prasideda value..
  • [atributas$=value]
    • - atributas, kurio reikšmė pasibaigia value..
  • :active - aktyvi (po paspaudimo) nuoroda
  • :link - paprasta/nelankyta nuoroda
  • :visited - lankyta nuoroda
  • element:emty - tuščias elementas
  • ::marker - sąrašo elementų žymės
  • :not(selecotr) - parenka viską, išskyrus nurodytą elementą
  • :first-child - parenka pirmąjį vaikinį elementą
  • :last-child - parenka paskutinį vaikinį elementą
  • :onyl-child - parenka
  • :first-of-type - parenka pirmąjį nurodyto tipo elemetą
  • :last-of-type - perenka paskutinį nurodyto tipo elementą
  • :only-of-type - parenka vienintelį nurodyto tipo elementą
  • :nth-child(n) - parenka nurodytą/us vaikinį/ius elementą/us
  • :nth-last-child(n) -
    • nuo galo parenka nurodytą/us vaikinį/ius elementą/us
  • :nth--typechild(n) - parenka nudodytą/us elementą/us pagal tipą
  • :nth-last-of-type(n) - parenka nuo galo nurodytą elementą/us pagal tipą
  • ::before - sukuria turinį prieš elementą
  • ::after - sukuria turinį už elemento
  • input:checked - pažymėtus radio/checkbox/options pasirinkimus
  • input:default - iš anksto prinktas pasirinkimas
  • input:enabled - input'as, kuris turi disabled atributą
  • input:enabled - input'as, kuris turi enablead atributo
  • input:focus - input'as, ant kurio yra paspausta
  • input:in-range - input'as, kurio vertė yra nurodytose ribose
  • input:out-of-range - input'as, kurio vertė yra už nurodytų ribų
  • input:inder - imput'as, kurio reikšmė neapibrėžta
  • input:invalid - input su netinkama verte
  • input:valid - input, kurrio vertė tinkama
  • input:required - input su required atributu
  • input:optional - input be required atributo
  • input:read-only: input su readonly atributu
  • input:read-write - input be readonly atributo
  • input::placeholder - įvesties laukelio placeholder atributo reikšmės parinkimas stilizavimui
  • element::first-letter - parenka elemento pirmąją raidę
    • element::first-line - parenka elemento pirmąją eilutę
  • ::selection - parenka vartotojo pažymėtą informaciją
